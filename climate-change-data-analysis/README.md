# Climate Change Data Analysis

## Project Overview
This project analyzes historical global temperature data to identify long-term climate change trends.

## Objectives
- Analyze global temperature trends over time
- Compare average temperatures across countries
- Visualize geographic climate impact at city level

## Tools Used
- Excel (data cleaning)
- SQL (analysis logic)
- Power BI (data visualization)
- ArcGIS (geographic mapping)

## Key Insights
- Global average temperatures show a steady increase since 1950
- Warming trends are consistent across multiple countries
- Urban regions show higher temperature concentrations
